# Estructura
- El proyecto usa Docker en el backend ya que es de mayor importancia guardar las dependencias. El proyecto del frontend se encuentra realizado en Vite con la versión de npm 20.20


- Solo es necesario correr el comando docker-compose up --build desde la raiz del repositorio para que cree y monte la imagen del backend y el comando npm run dev en la carpeta del frontend.


- Se agregaron varios usuarios con los siguientes códigos para la prueba de validación de código y de sesiones activas e inactivas: AAA001, AAA02 y AAA03
